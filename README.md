# React API Integration Project

## Overview

This project shows how to fetch and display data in React using local JSON, Fetch API, and Axios, along with routing and filtering.

## Features

* Local JSON data fetching
* Public API integration (Fetch)
* Fake API integration (Axios)
* Dropdown filtering
* Navigation using React Router
* Material UI buttons

## Tech Stack

React, Fetch API, Axios, React Router, Material UI

## Run Project

```bash
npm install
npm start
```
